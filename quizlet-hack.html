Getting Started


Welcome to the Quizlet Hack! This simple two-step process will have you up and running in no time. Click "Next" to begin.


Hack by: thelazysquid

Guide by: CodeMark

Step 1: Download Tampermonkey


To modify user scripts, we'll need Tampermonkey. Follow these simple steps:


  • Click on this link to install Tampermonkey.
  • Add the extension to Chrome.

Once installed, you'll have the power to customize your Quizlet experience!

Step 2: Install the Quizlet Hack Script


Now that you have Tampermonkey, let's add the Quizlet Hack script:


  • Go to this link: Script.
  • Click "Install" to add the script to Tampermonkey.
  • Feel free to reinstall the script if you ever encounter any issues or want to update it.

To open/close the hud, press "\" (backslash) on your keyboard while on Quizlet. It will show various menus, which should be fairly self explanatory.

Enjoy the Quizlet Hack!
